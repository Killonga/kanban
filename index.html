Proyecto Destacado 2026

KanbanPro

Aplicación fullstack de gestión de tareas estilo Kanban con autenticación segura de usuarios y persistencia real en base de datos. Permite organizar el trabajo en tableros con columnas (Backlog, Doing, Review, Done) y tarjetas que se pueden crear, editar, priorizar y asignar. Construida de cero, sprint a sprint, hasta un MVP completamente funcional.

Desarrollo por sprints

Sprint 1 — Prototipo funcional

SSR con Handlebars, navegación base, persistencia en archivos JSON, sesiones con express-session.

Sprint 2 — Arquitectura con PostgreSQL

Modelos Sequelize (Usuario, Tablero, Lista, Tarjeta), relaciones hasMany/belongsTo, seed script y test de CRUD aislado.

Sprint 3 — API RESTful & Seguridad

JWT + bcryptjs, endpoints CRUD protegidos con middleware, Eager Loading con Sequelize, conexión completa al dashboard.

Tecnologías

Node.js Express 5 Handlebars PostgreSQL Sequelize ORM JWT bcryptjs express-session REST API
